# Промпт для нейросети: UI/UX «Структуратор» (B2B/B2G)

## Текст промпта

**Задача по UI/UX дизайну (специфика B2B/B2G и полевых сотрудников):**

Напиши конфигурацию `ThemeData` (Material 3) для приложения **«Структуратор»** и примени её ко всем экранам. Интерфейс для корпоративного и госсектора: сотрудники «в полях», работа на солнце и в плохом освещении, длительное чтение без усталости глаз.

### 1. Цветовая палитра (`ColorScheme`)

| Роль | Значение | Назначение |
|------|----------|------------|
| `primary` | `Color(0xFF1565C0)` | Глубокий синий — навигация, главные кнопки, FAB |
| `scaffoldBackgroundColor` | `Color(0xFFF5F7FA)` | Off-white фон (не ослепляет на яркости) |
| Карточки (`surfaceContainerLowest`) | `Color(0xFFFFFFFF)` | Белые карточки на сером фоне |
| Текст | `Color(0xFF263238)` | Графитовый, не чистый `#000000` |
| `error` | `Color(0xFFD32F2F)` | Приглушённый красный — только критические ошибки |
| Warning (черновик, оффлайн) | `Color(0xFFFF8F00)` / `Color(0xFFE65100)` | Янтарь/оранжевый — статусы «черновик», «ожидает» |
| Success | `Color(0xFF2E7D32)` | Спокойный зелёный — «сгенерировано», «принято», «отправлено» |
| Шапки / AppBar на градиенте | `Color(0xFF1A2B4A)` → `#1565C0` | Navy → deep blue |

### 2. Типографика

- Шрифт: **Roboto** (системный sans-serif).
- Межстрочный интервал `height`: **1.4–1.5** для `bodyLarge`, `bodyMedium`, подписей.
- Заголовки `titleLarge`, `headlineSmall`: **FontWeight.w600** (не w800).
- Вторичный текст: `onSurfaceVariant` ≈ `#546E7A`.

### 3. UX для полевой работы

- **Tap targets:** минимум **48×48 dp** для кнопок, `IconButton`, пунктов списка (`AppTheme.minTouchTarget`).
- **Карточки:** каждый отчёт в `Card` / `Material` с **elevation 1–2**, белый фон, скругление 14 dp.
- **Индикация статуса:** цветной круг 10 dp + Chip:
  - зелёный — сгенерирован / принят;
  - оранжевый — черновик, на проверке;
  - красный — отклонён / ошибка.
- **Кнопки:** главное действие (`Сгенерировать`, `Отправить`, FAB) — **FilledButton** с `primary` (синий); отмена/назад — **OutlinedButton** или **TextButton**.

### 4. Файлы проекта

- Тема: `lib/config/app_theme.dart`
- Общие виджеты: `lib/widgets/app_ui.dart`, `lib/widgets/report_chips.dart`
- Статусы: `lib/models/report_status.dart`
- Подключение: `lib/main.dart` → `theme: AppTheme.light`

### 5. Ограничения

- Material 3, без Firebase.
- Не использовать чистый белый `#FFFFFF` как фон всего экрана.
- Не делать primary оранжевым — оранжевый только для warning/черновиков.
- Сохранить градиентные шапки (`AuroraHeader`) для узнаваемости бренда хакатона.

## Реализация в репозитории

Палитра и `ThemeData` уже внедрены в `lib/config/app_theme.dart`. При доработке экранов сверяйтесь с этим документом и не переопределяйте `FilledButton.styleFrom` на оранжевый без причины.
